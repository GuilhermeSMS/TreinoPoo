# ☕ Sistema de Funcionários - POO Java

Projeto de estudo de Programação Orientada a Objetos em Java.

## 📚 Sobre o Projeto

Hierarquia de classes com cálculo de salário polimórfico.

- Classe abstrata `Funcionario`
- Subclasses: `Gerente`, `Desenvolvedor`, `Estagiario`
- Cálculo de salário polimórfico por tipo de funcionário
- Total da folha de pagamento via `List<Funcionario>`

## 🛠️ Tecnologias Utilizadas

- **Linguagem:** Java
- **IDE:** IntelliJ IDEA
- **Gerenciador de Dependências:** Maven
- **Versionamento:** Git
